# GameTV
游戏直播平台的快捷入口插件（webkit内核浏览器插件）。

目前已知支持浏览器：谷歌chrome浏览器、QQ浏览器、百度浏览器。

不会安装插件，[请点击这里查看攻略！](http://jingyan.baidu.com/article/da1091fbdf12e9027949d673.html)

闲的无聊做了这个插件，主要为了方便自己。我的标签页安装了其他插件，起冲突了。而我的收藏夹都要爆了，太多了一个一个找不方便。
